# Мой проект об окончании второго курса

Данный проект представляет собой веб-приложение женского фитнес клуба. Данное приложение имеет бэкенд, фронтенд и базу данных

## Стек технологий

- Фронтенд: React + TS
- Бэкенд: Java Spring Boot + Hibernate
- Proxy: Go + Gin + yl2chen cidranger
- База данных SQL: PostgreSQL
- База данных NoSQL для proxy: MongoDB
